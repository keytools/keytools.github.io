Description

Online tool that will help you learn Touch Typing (also called Blind Typing) which is a style that let you type without using your eyesight to find the keys on the keyboard. Every time you look at the keyboard or make a mistake, you slow down you typing speed. So to be more productive, it would be nice if you could type fast and accurate and this what we will help you do in this website.

Many apps have been developed to help users improve their typing skills. Our difference with the other applications is that you can improve your typing skills while writing beautiful Quotes and Poems and if you are a Software Engineer like me, you can write Java or Python source code. Another important feature of the site is the ability to view your typing speed and accuracy and how you have improved. Each typing test score will be saved and you can see graphs of your progress. Last but not least, you can add your own text to type by clicking the icon present in the navbar above.

The convention on how we should use our keyboard is to put our fingers on the Home Row, the one containing the F and J keys. There are small bumps on these keys and you should put your index fingers on the bumps. Each finger is responsible for its own set of keys, as shown in the image bellow:

The majority of people types between 35 and 40 words per minute (WPM) which is equivalent to 190 and 200 characters per minute (CPM). However, professional typists type a lot faster exceeding 75 WPM. Typing speed generally improves with practice. While practicing, it is important to ensure that there are no weak keys. Typing speed is typically determined by how slow these weak keys are typed rather than how fast the remaining keys are typed. That's why it is advisable to start by our Training lessons to memorize keys position. By default, the Qwerty keyboard is selected and if you are using Azerty keyboard, you can change your preferences to obtain the adequate lessons.
